JsonApi Example
===============

> An example of json-api application in F#

---

## Development
### Requirements
- [dotnet core](https://dotnet.microsoft.com/learn/dotnet/hello-world-tutorial)
- [FAKE](https://fake.build/fake-gettingstarted.html)

### Build
```bash
./build.sh
```

### Run
```bash
./build.sh -t Run
```

### Watch
```bash
./build.sh -t Watch
```
